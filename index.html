Team Photos

Drop JPG/PNG files into /pics/ in your repo and list them in /pics/gallery.json (see format below). They will appear here with a lightbox.

How to add photos
  1. Create a folder pics/ at the repo root.
  2. Add your images there (e.g., team1.jpg, goal2.png).
  3. Create pics/gallery.json like this: 
    {
  "images": [
    {"src": "pics/team1.jpg", "alt": "First practice"},
    {"src": "pics/game1_goal.jpg", "alt": "Game 1 goal"}
  ]
}
